COMPACS RAG Desktop — быстрый старт
====================================

1. Распакуйте Part1–Part3 по ASSEMBLE.md в одну папку.
2. Дважды щёлкните START.cmd (консоль) или START_UI.cmd (окно).
3. Задайте вопрос. Браузер для консольного режима не нужен.

Все пути относительные. Python, Docker, Ollama и внешние API не требуются.
START.cmd поднимает llama-server (:8081 embed, :8082 chat) и запускает main.exe.

Что лежит в папке после сборки комплекта:
  main.exe
  config.yaml
  vectors.bin
  lemma_map.tsv
  llama\llama-server.exe (+ DLL)
  models\nomic-embed-text.gguf
  models\llama3.2-3b-instruct-q4_K_M.gguf
  assets\index.html
  START.cmd / START_UI.cmd / STOP.cmd
  RAG_ARCHITECTURE.md

Порты:
  8765  — UI/API (только START_UI)
  8081  — эмбеддинги (llama-server)
  8082  — генерация ответа (llama-server)

Остановка: STOP.cmd

Требования:
  Windows 10/11 x64
  WebView2 Runtime (обычно уже есть с Edge; только для START_UI)
  ~2.5 GB свободного места

Пересборка архивов из исходников (на машине разработчика):
  build.bat
  powershell -ExecutionPolicy Bypass -File .\pack_parts.ps1
